Guide des formations aux métiers de
l'environnement en région P.A.C.A.




EPUISE....
 

C'est le premier document d'information complet sur les formations aux métiers de l'environnement dans la région P.A.C.A. Il répertorie 232 formations sous forme de fiches individuelles, complétées de plusieurs index de classement.
Vous pourrez ainsi connaître :
 - le niveau de qualification requis pour chaque formation
- la durée de l'enseignement
- le type d'enseignement
- les débouchés
- les orientations professionnelles possibles

Les formations proposées peuvent être initiales, continues, qualifiantes ou diplômantes.

Partenaires techniques et financiers :

  • Conseil Général 13
  • Conseil régional PACA
  • DIREN
